# Build: Rewrite /hive template with four sections

## Files Changed

### `site/graph/store.go`
- Added `ListHiveAgentTasks(ctx, actorID, limit)` — returns up to N open tasks authored by agents (BOUNDED). Mirrors `GetHiveCurrentTask` pattern but returns `[]Node`.

### `site/graph/handlers.go`
- Added `"Reflector"` / `"[hive:reflector]"` to `pipelineRoleDefs` (fourth pipeline phase).
- Added `maxHiveAgentTasks = 10` constant.
- Added `parseIterFromPosts(posts []Node) int` — extracts highest `iter N` number from post titles using regexp.
- Updated `handleHive` to call `ListHiveAgentTasks` instead of `GetHiveCurrentTask`, pass `iterCount`, and use new `HiveView` signature.

### `site/graph/views.templ`
- Rewrote `HiveView` signature: `tasks []Node` replaces `currentTask *Node`; added `iterCount int` parameter.
- **Section 1 — Header:** "The Civilization Engine" headline, subtitle, iteration counter badge (when `iterCount > 0`), plus stat cards (Features shipped / Total autonomous spend / Avg cost).
- **Section 2 — Pipeline status:** Four phase cards in a 2×2 / 4-column grid (Scout / Builder / Critic / Reflector). Active phase uses rose border (`border-brand/40`) and rose name text; idle uses default edge.
- **Section 3 — Current work:** List of open agent tasks with title, state badge, assignee. Shows "Idle" dot when no tasks.
- **Section 4 — Build log:** Recent posts with title, body preview (120 chars), and relative timestamp.
- Main content `<main>` element gets `id="hive-content"` for HTMX swap target.
- Added `hivePipelineCardClass` and `hivePipelineNameClass` helpers.

### `site/graph/views_templ.go` (generated)
- Regenerated by `templ generate`.
